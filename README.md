# 🕵️ Πόιος είναι ο «Μυστικός» – Εκπαιδευτικό Παιχνίδι Κρυφού Ρόλου (impostor‑style)

*Ένα διαδραστικό, προσβάσιμο, εκπαιδευτικό παιχνίδι κρυφού ρόλου για τάξεις, ομάδες και εκπαιδευτικές δραστηριότητες, με επεξεργασία κατηγοριών, τυχαία σειρά ομιλίας και αυτόματη βαθμολογία.*

---

## 📌 Περιγραφή

Το **«Πόιος είναι ο Μυστικός;»** είναι μια διαδικτυακή εφαρμογή βασισμένη στη λογική του **impostor‑style hidden‑role game**, προσαρμοσμένη για **εκπαιδευτικό πλαίσιο**. Οι παίκτες λαμβάνουν μια λέξη ή ρόλο, συζητούν και στη συνέχεια ψηφίζουν ποιος πιστεύουν ότι είναι ο «μυστικός» (impostor).

**Χαρακτηριστικά:**
- ✅ Πλήρως στα ελληνικά, φιλικό για σχολική χρήση
- ✅ Προσβάσιμη διεπαφή (ARIA, keyboard navigation)
- ✅ **Στούντιο κατηγοριών**: δημιουργία, επεξεργασία, ομαδοποίηση λέξεων
- ✅ **Βοηθητικές λέξεις** για impostor (μορφή: «κύρια,βοηθητική»)
- ✅ Τυχαία σειρά ομιλίας και διαδραστική ψηφοφορία
- ✅ Αυτόματη βαθμολογία με δυνατότητες εξαγωγής/διαγραφής
- ✅ **Ομαδοποίηση κατηγοριών** (γονικές ομάδες) για εύκολη πλοήγηση
- ✅ Εξαγωγή/εισαγωγή ρυθμίσεων σε JSON
- ✅ Responsive design – λειτουργεί σε κινητά, tablet, desktop
- ✅ Ανοιχτό API (δεν απαιτείται σύνδεση, όλα local)

---

## 🚀 Τεχνολογίες

- **Frontend:** Καθαρό HTML5, CSS3, JavaScript (ES6+)
- **Storage:** `localStorage` για αποθήκευση κατηγοριών, βαθμολογίας, ρυθμίσεων
- **Εικόνες/Assets:** SVG, PNG (φιλικά favicons, logos)
- **Πρόσβαση:** ARIA labels, semantic HTML, keyboard navigation
- **Hosting:** Στατικό hosting (όλα τα αρχεία είναι αυτόνομα)

## 🎮 Πώς Λειτουργεί

### 1. **Ρυθμίσεις Παιχνιδιού**
   - Επιλογή αριθμού παικτών (3‑20)
   - Επιλογή αριθμού impostors (προσαρμόζεται αυτόματα)
   - Προαιρετικά ονόματα παικτών
   - Επιλογή κατηγορίας από αναδυόμενο στούντιο

### 2. **Στούντιο Κατηγοριών**
   - **Αναζήτηση** και **φιλτράρισμα** ανά ομάδα
   - **Δημιουργία νέας κατηγορίας** (όνομα, γονική ομάδα, λίστα λέξεων)
   - **Επεξεργασία** υπαρχουσών κατηγοριών (ονόματα, λέξεις, ομάδες)
   - **Διαχείριση γονικών ομάδων** (π.χ. «Πληροφορική», «Τεχνολογία», «Γενικά»)
   - **Εξαγωγή/Εισαγωγή** όλων των ρυθμίσεων σε JSON

### 3. **Μοιρασιά Ρόλων**
   - Κάθε παίκτης πατάει στην κάρτα για να δει:
     - **Αθώος:** τη **κύρια λέξη**
     - **Impostor:** είτε «❌ impostor» είτε **βοηθητική λέξη** (αν ενεργοποιημένο)

### 4. **Συζήτηση & Ψηφοφορία**
   - Εμφάνιση **τυχαίας σειράς ομιλίας**
   - Διαδραστική ψηφοφορία (ένα κλικ ανά παίκτη)
   - Αποτελέσματα με **αποκάλυψη impostor και λέξης**

### 5. **Βαθμολογία**
   - **Αθώοι βρίσκουν impostor:** +10 πόντοι σε κάθε αθώο
   - **Impostors κερδίζουν:** +20 πόντοι σε κάθε impostor
   - Αποθηκεύεται αυτόματα (`localStorage`)
   - Δυνατότητα **εξαγωγής CSV** και **διαγραφής**

## ⚙️ Ρυθμίσεις & Δυνατότητες

### 📦 Κατηγορίες & Ομάδες
- **Προεπιλεγμένες κατηγορίες:** Προγραμματισμός, Δίκτυα, Βάσεις Δεδομένων, Ασφάλεια, Υλικό, Ιστός, Τεχνολογία, Ελεύθερο Θέμα
- **Ομαδοποίηση:** Κάθε κατηγορία ανήκει σε μια **γονική ομάδα** (π.χ. «Πληροφορική»)
- **Δυνατότητα προσθήκης** απεριόριστων κατηγοριών και ομάδων

### 🔄 Βοηθητικές Λέξεις
- Μορφή: `κύρια,βοηθητική` (π.χ. `Σκύλος,τετράποδο`)
- Αν ενεργοποιηθεί ο διακόπτης, ο impostor βλέπει **μόνο τη βοηθητική λέξη**
- Στατιστικά εμφανίζονται κάτω από τον διακόπτη

### 📊 Βαθμολογία & Αποθήκευση
- **Αυτόματη αποθήκευση** μετά από κάθε παιχνίδι
- **Εξαγωγή CSV** για εκτύπωση ή ανάλυση
- **Διαγραφή όλων** των βαθμών με ένα κλικ
